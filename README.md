# Sales-Dashboard-Power-BI

An interactive Excel-based Sales Dashboard built to help business stakeholders track sales performance, profitability and key trends across products, categories and payment channels — enabling faster, data-driven decision-making.

![image](https://github.com/nethmi-i-j/Sales-Dashboard-Power-BI/blob/9ad698d2e2957158863e554c07838bd90fc7714a/Sales_Dashboard_.JPG)

## Objective

Retail businesses often struggle to get a quick, unified view of sales performance across products, categories and some other factors. This dashboard consolidates raw transactional data into a single interactive view, helping stakeholders:

* Identify top-performing products and categories
* Track profit margins across sale types and payment modes
* Monitor daily/monthly sales trends
* Filter data dynamically by year, month, sale type and payment mode

## Data Structure

* Input Data – Transaction-level records (Date, Product ID, Quantity, Sale Type, Payment Mode, Discount %)
* Master Data – Product reference table (Product Name, Category, UOM, Buying Price, Selling Price)  

## Tools & Techniques Used

* Power Query – Created total buying value and total selling value column
  
* Data Modeling – Relational structure linking transaction data (Input Data) with a product master table (Master Data) via Product ID
  
* DAX Measures – KPIs for Total Qunatity, Total Sales, Profit, and Profit %
  
* Charts for Visualization – Bar, donut, and line charts for trend and category analysis
  
* Slicers – Interactive filters for Year, Month, Sale Type, and Payment Mode
  
* Dashboard Design – Clean interactive layout with consistent color theme and KPI cards

## KPIs

* Total Sales 
* Quantity  
* Profit 
* Profit %

## Interactive Charts

* Sales & Profit by Month 
* Sales by Type (Direct Sales, Online, Wholesaler)
* Sales by Product
* Sales by Category
* Sales by Payment Mode (Online vs Cash)
* Daily Sales Trend
